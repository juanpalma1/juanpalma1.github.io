an open-source serverless framework that offers webhooks as a service.

  • Learn More

Tacklebox includes a CLI tool, a RESTful API, 4 client libraries and a management UI

The CLI tool makes it easy to deploy and tear down AWS infrastructure that offers webhooks as a service, the client libraries are written in JavaScript, Ruby, Python and Go, and you can run the management UI locally with one command.
